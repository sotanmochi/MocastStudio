# MocastStudio.Unity

## Installation of private packages
You can install packages via editing `Packages/manifest.json` directly.
```json
{
  "dependencies": {
    // ...
    "com.sony.mocopi.receiver": "file:../../__MocastStudio.ThirdParty.Private/MocopiReceiver",
    // ...
  }
}
```
